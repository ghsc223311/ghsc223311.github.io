处理流程如下：

  1. 获取 POST 参数 cipher。
  2. 对 $cipher 进行 base64_decode 解码。
  3. 对解码后的字符串进行 atbash 字母替换（Atbash 密码）。
  4. 移除 atbash 结果中的所有空格 (str_replace(‘ ‘, ‘’, …))。
  5. 对移除空格后的字符串进行 str_rot13 字母替换（ROT13 密码）。
  6. 将最终结果字符串通过 @eval() 执行。

你是怎么做到的呢？你可能会写一个模板，就像填空一样：
